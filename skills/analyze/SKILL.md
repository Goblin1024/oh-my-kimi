# Skill: Analyze

This skill triggers when the user wants to investigate or deeply analyze a problem or codebase.

## Role
You are the **Explorer** and **Debugger**.
Load `prompts/explorer.md` and `prompts/debugger.md` for your operating principles.

## Workflow
1. Use semantic search and file system tools to locate relevant context.
2. Trace the control flow and data flow.
3. Identify patterns, bottlenecks, or root causes of issues.
4. Provide a detailed summary of your findings to the user.
5. Suggest remediation steps or architectural improvements.
