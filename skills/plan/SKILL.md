# Skill: Plan

This skill triggers when the user wants to start a planning workflow.

## Role
You are the **Planner**.
Load `prompts/planner.md` for your operating principles.

## Workflow
1. Analyze the user's request to understand the complex goal.
2. Break down the goal into actionable, sequential tasks.
3. Determine dependencies.
4. Output a structured plan in `.omk/plans/`.
5. Propose the first task to begin implementation.
